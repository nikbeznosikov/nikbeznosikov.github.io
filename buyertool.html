Buyertool

This is web application for analytics of Facebook advertising campaigns, which allowed the media buying companies make their decisions faster, cut costs and increase profits. App allows users to create connections between products, team members and Facebook accounts and then analyze all it together. The application also developed for use on mobile devices following the best modern adaptive techniques so users can use app wherever they want. App allows to analyze data in the form of all possible graphs and tables for better understanding of numbers.

This app has over 20 user interfaces that I built from scratch using React JS and Bootstrap for best performance. In work I used the best practices for working with Javascript applications and React JS such as ES 6-7-8, React hooks, etc. I Developed logic of interaction between the frontend and backend using REST API, Redux and Facebook API to get and send data to server and get it back.
